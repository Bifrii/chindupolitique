# Chindupolitique - Project Context for Agents

## Stack
- Frontend: built with Lovable (React/Next.js)
- Database: see project files for schema
- Hosting: Vercel (auto-deploy on push to main)
- Repo: https://github.com/Bifrii/chindupolitique

## Agent instructions
- Always read existing code before making changes
- Never break existing functionality
- Always create a feature branch before editing
- Write clear commit messages: fix/feat/refactor + short description
- After changes, summarise what was done for the reporter agent
